# ゲームデザイン仕様索引

この索引は、ネオン・アンダーレルムTRPGのゲームデザイン上の仕様を参照する入口である。

サイトのUI・ページ・変換・実装に関する要件は `docs/requirements.md`、画面のデザイン正本は `docs/design/`、ExcelからJSONへの変換仕様は `docs/conversion/` に置く。ゲーム上の用語、データの意味、取得や使用の規則はこの索引と `docs/game-design/` 配下で扱う。

## 仕様ファイル一覧

| ファイル                     | 扱う内容                             | 主な参照場面                   |
| ---------------------------- | ------------------------------------ | ------------------------------ |
| `docs/game-design/skills.md` | スキルが持つゲーム上の情報と表記規則 | スキルデータの設計、ルール確認 |

## 分割方針

ゲーム上のデータ種別ごとにファイルを分ける。表示方法、Excel列名からJSONへの変換、Component APIは、この索引では定義しない。
