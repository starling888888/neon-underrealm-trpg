# 27-3-user-directed-content-revisions

## 目的

ユーザー指定のコンテンツ修正として、`/world` にv1.0ルールブックの勢力を情報を落とさずに追加し、`/data` のスキル凡例の直後に同名スキルと攻撃スキル効果の補足を追加する。

## 背景

現在の`/world`は、v1.0ルールブックの「勢力」にある政治、企業、裏社会、独立勢力の説明を掲載していない。一方で、既存の「街を動かす勢力と人物」節は著名なNPCの紹介を担っているため、勢力説明とNPC紹介を別節に分ける必要がある。実装後の全体照合で、01.ルールブックのワールドガイドにある見出し構造、具体例、因果関係、断定の強さの一部が欠落または変化していることも確認されたため、勢力以外の本文も同じ基準で修正する。

`/data`のスキル凡例には、`「スキル名：XX」`の同名扱いと、攻撃スキルのダメージ決定以降の効果がリアクション成功時に発揮されない条件を補足する。

このタスクはユーザー指定の計画外コンテンツ修正である。ユーザー指示により`docs/plan.md`へ計画項目は追加しない。

関連参照:

- `.raw/v1.0/01.ルールブック.md` の「ワールドガイド」全体
- `docs/requirements/pages.md` の`/world`および`/data`
- `docs/requirements/data-display.md` のFR-04-01、FR-04-03
- `docs/design/world/notes.md`

- `docs/TODO.md`のNPC個別画像・Excel / JSON管理の将来対応

## 対象範囲

- `.raw/contents/world.md`と`src/pages/world.mdx`を、`contents-markdown-authoring`に従って同じ内容へ更新する。
  - 節順を`歴史`、`仕事人`、`科学技術`、`信仰`、`勢力`、`著名なNPC`、`強大な敵`、`この街で生きる`とする。
  - 01.ルールブックのワールドガイドにある見出し構造、具体例、因果関係、限定・断定の強さを、文章を整えても省略または変更しない。科学技術にある各`表の顔`、`裏の顔`も見出しレベルを問わず維持する。
  - ページ冒頭文、勢力の導入文、`この街で生きる`はサイト用の追加本文として維持する。強大な敵には01.ルールブックにない補足文を追加しない。
  - 無料版ルールブックをワールドガイド本文の比較元にしない。
  - v1.0ルールブックの勢力として、政治、企業（クロスバース、ノギサカインダストリアル、黒紅公司、T&Jケミカル）、裏社会（互助組合ヨド、ブルーブラッド、アシカガ組）、独立勢力（副都警666課、組合、一つの日本）をすべて記載する。
  - 政治と10組織のそれぞれについて、v1.0ルールブックにある所属、支配状況、活動内容、具体例、因果関係を本文で説明する。文章を整えても情報を省略または一般化せず、見出しまたは組織名の列挙だけにはしない。
  - 現在の「街を動かす勢力と人物」節を`著名なNPC`へ変更し、既存の`NpcCard`紹介をこの節に置く。企業・ヤクザ・警察などが街を動かすという旧導入文は残さず、著名なNPCを紹介する導入文へ差し替える。
  - 既存の`NpcCard`のprops、人物、個別カードの見た目は変更しない。
- `.raw/contents/data.md`と`src/pages/data/index.mdx`を、`contents-markdown-authoring`に従って同じ内容へ更新する。
  - スキル凡例と「カードの項目」を含む`skill-legend-layout`の直後、`タイミング`節の直前に、次の文を追加する。

    > 「スキル名：XX」という記載があった場合、XXには任意の文字が入ります。「スキル名：XX」と言う名称のスキルはすべて同名のスキルとして扱い、2つ以上同時に取得できません。別のスキルとして扱う場合は効果に記載があります。

  - 上記の文の直後に、既存の`Callout`を`type="warning"`、titleを`攻撃スキルの効果`として置く。本文では、攻撃スキルの効果でダメージ決定以降に効果を発揮するものは、リアクションに成功された場合に効果を発揮しないことを説明する。
  - Callout内に順序なしリストを置き、次を明記する。
    - 判定数または気合獲得値を上昇させる効果は、攻撃の成否にかかわらず発揮される。
    - ダメージまたは攻撃力を上昇させる効果、毒のようなバッドステータス、判定ペナルティを与える効果は、リアクションに成功されたら発揮されない。
    - コンボ増加値を上昇させるなど次の攻撃を強化する効果、次の味方キャラクターの攻撃力を上げるなど続く攻撃に影響を及ぼす効果は、リアクションに成功されたら発揮されない。
- `docs/issue/27-3-user-directed-content-revisions.md`に、実施済みの確認だけを記録する。

## レビュー観点

- ワールドガイド全体が01.ルールブックの見出し構造と本文情報を漏れなく維持し、指定された節順になっているか。無料版ルールブック由来の情報を混ぜていないか。
- `勢力`節が政治、企業4、裏社会3、独立勢力3を漏れなく含み、NPC紹介と混在していないか。
- `著名なNPC`節への変更後も、既存NPCカードの人物、二つ名、説明、表示順が保たれているか。
- `「スキル名：XX」`注記とwarning Calloutが、凡例とカード項目の直後かつタイミング節の直前に置かれているか。
- warning Calloutの例外と不発条件が、ユーザー指定の3分類と矛盾していないか。
- `docs/design/world/`と`docs/design/data/`の既存designに対し、見出し追加と既存Calloutの利用だけで足り、design画像の新規作成を要しない判断が妥当か。
