# 共通スキル

プライマリ流儀、生き様を問わず取得できるスキルです。経験点5点で取得できることに注意してください。ただし取得できるレベルの上限は **格(プライマリ流儀レベルと生き様レベルの合計)の半分** までです。

## 自動習得スキル

全てのPCはキャラクター作成時に以下の `基本の一撃` を自動習得します。経験点を支払う必要はありません。

<CardContainer>
  {commonSkills.bonus.map((skill) => <SkillCard {...skill} anchorId={skill.id} />)}
</CardContainer>

## 初期作成時から取得可能なスキル

<CardContainer>
  {commonSkills.basic.map((skill) => <SkillCard {...skill} anchorId={skill.id} />)}
</CardContainer>
