仕事人の戦い方、使えるスキル、装備を確認するためのデータをまとめています。キャラクターを作るときは、まず流儀と生き様を選び、必要に応じて共通スキルとアイテムを参照してください。

## データを探す

- <InternalLink href="/data/ryugi">流儀</InternalLink>: 仕事人としての戦い方と、流儀ごとのスキルを確認します。
- <InternalLink href="/data/ikizama">生き様</InternalLink>: どのように生き、どの技術を使うかを選びます。
- <InternalLink href="/data/common-skills">共通スキル</InternalLink>: どの仕事人でも取得できるスキルを確認します。
- <InternalLink href="/data/items">アイテム</InternalLink>: 武器、防具、お守り、サイバネ、ナノマシン、ドラッグを確認します。

初めてPCを作る場合は、<InternalLink href="/character-making">キャラクターメイキング</InternalLink>で作成手順を確認してから、必要なデータへ進んでください。

## スキルの見方

スキルカードには、使える場面、必要なコスト、判定に使う技能、取得や使用の条件がまとまっています。効果の詳しい処理は、カード本文と<InternalLink href="/rules/battle">戦闘</InternalLink>のルールを合わせて確認してください。

### カードの項目

1. **名称**: スキルの名前です。同じスキルは、本文に特記がない限り複数取得できません。
2. **最大LV**: そのスキルを成長させられる上限です。
3. **タイミング**: 戦闘中にスキルを使える場面です。略号の意味は次の表で確認します。
4. **コスト**: 使用時に支払う気合、体力、精神力、小銭、または覚悟です。コスト欄の`SP`は、支払う点数で効果が変わることを表します。
5. **技能**: 判定を行うときに使う技能です。`能動`は攻撃時の技能、`受動`はリアクション時の技能を表します。
6. **取得制限**: 取得するために必要な条件です。
7. **使用制限**: 使用回数や、使用できる条件です。
8. **対象**: 攻撃や効果を受けるキャラクターや範囲です。
9. **射程**: 攻撃や効果を届かせられる距離です。
10. **効果**: スキルができることと、その詳しい処理です。`lv` はそのスキルの取得レベルを表します。

「スキル名：XX」という記載があった場合、XXには任意の文字が入ります。それらは同名のスキルとして扱います。例えば「命中強化：清林」と「命中強化：大山」という名称のスキルは同名のスキルとして扱い、同時に取得できません。別のスキルとして扱う場合は効果に記載があります。

<Callout type="warning" title="攻撃スキルの効果">
攻撃スキルの効果でダメージ決定以降に効果を発揮するものは、リアクションに成功された場合に効果を発揮しません。

- 判定数を上昇させたり、気合獲得値を上昇させる効果は、攻撃の成否にかかわらず発揮されます。
- ダメージや攻撃力を上昇させたり、毒のようなバッドステータスや判定ペナルティを与える効果は、リアクションに成功されたら発揮されません。
- コンボ増加値を上昇させるような次の攻撃を強化する効果、次の味方キャラクターの攻撃力を上げるような続く攻撃に影響を及ぼす効果は、リアクションに成功されたら発揮されません。
</Callout>

### タイミング

`○-○`のように記号を`-`でつないだ表記は、コンボ攻撃中に使うスキルのタイミングです。コンボのどの攻撃で使えるかを表します。記号の代わりに`A` が書いてある場合は、`○`、`×`、`☆`のいずれかを表します。

| 表記 | 使う場面 |
| --- | --- |
| `Pv` | 常に効果を発揮します。 |
| `SU` | ラウンドの準備をするセットアップで使います。 |
| `INI` | 次に行動するキャラクターを決めるときに使います。 |
| `CU` | ラウンドを終えるクリンナップで使います。 |
| `M` | 自分の手番に、1行動として使います。 |
| `○-○` | コンボ中のどの攻撃でも使えます。 |
| `×-A`、`☆-A` | コンボの最初の攻撃でだけ使えます。 |
| `A-×`、`A-☆` | コンボの最後の攻撃でだけ使えます。 |
| `R` | 攻撃へのリアクションで使います。 |
| `Aa` | 攻撃の判定直後に使います。 |
| `Ra` | リアクションの判定直後に使います。 |
| `D` | ダメージを決定するときに使います。 |
| `SP` | スキル本文で指定された場面に使います。 |

攻撃、リアクション、ダメージ決定の詳しい流れは、<InternalLink href="/rules/battle">戦闘</InternalLink>で確認してください。

### コストと制限

コストの `気X`、`体X`、`精X` は、それぞれ気合、体力、精神力をX点支払うことを表します。`気SP`、`体SP`、`精SP` は、支払った点数によって効果が変わるコストです。具体的な支払い方と上限は、スキル本文と関連する流儀の説明を確認してください。また例外的に、`小銭X`や`小銭SP` のように小銭でコストを支払うスキルもあります。

覚悟は、特別なコストです。戦闘ルールにおける<InternalLink href={`/rules/battle#${kakugoHeadingId}`}>覚悟</InternalLink>と同じように縁を失うことで支払えます。今生の縁を覚悟コストにしても効果は変わりません。

取得制限は、そのスキルを取るための条件です。使用制限は、1ラウンドに何回使えるか、またはどの条件で使えるかを表します。カードで `-` と表示される項目には、そのスキルに該当するコスト、取得制限、使用制限、または射程がありません。

#### 取得制限

| 表記 | 条件 |
| --- | --- |
| `自動習得` | 特定の条件で自動習得する。 |
| `プライマリ限定` | 所属する流儀がプライマリ流儀でなければ取得できない。 |
| スキル名 | 記載されたスキルを最大レベルで取得していなければ取得できない。 |

#### 使用制限

| 表記 | 使用制限 |
| --- | --- |
| `N/R` | 1ラウンドにN回まで使用できる。 |
| `N/Sn` | 1シーンにN回まで使用できる。 |
| `N/Sc` | 1シナリオにN回まで使用できる。 |
| `特殊` | 使用制限を効果本文に記載する。 |

`N` には、数値の代わりに `LV` と記載できる。複数の使用制限を記載する場合は、`&` で結ぶ。

#### 対象

| 表記 | 対象 |
| --- | --- |
| `自身` | スキルを使用するキャラクター自身を対象とする。 |
| `N体` | N体のキャラクターを対象とする。 |
| `範囲N` | 範囲Nを対象とする。 |
| `シーン` | シーンに登場している任意のキャラクターを対象とする。 |
| `特殊` | 対象を効果本文に記載する。 |

`N体` および `範囲N` の `N` には、数値の代わりに `LV` と記載できる。

#### 射程

| 表記 | 射程 |
| --- | --- |
| 0以上の整数 | 自分から何マス離れているかを表す。 |
| `武器` | 使用する武器の射程を使う。 |
| `シーン` | シーンに登場している任意の対象を選べる。 |
| `特殊` | 射程を効果本文に記載する。 |

<Callout type="danger" title="特別なスキル">

**覚悟の力：XX**は、スキルLV3まで成長させないと効果を発揮しません。使えるのはシナリオに1回だけで、使用回数を回復させる効果でも再び使えるようにはなりません。

**帰還不能地点：XX**は、スキルLV3まで成長させないと効果を発揮しません。

</Callout>

### クロスコンボ

**タイミングが「`A-☆`」、「`☆-A`」のスキル**は、単体でも非常に強力な攻撃スキルです。二人のキャラクターが攻撃をつなげると、さらに強力なクロスコンボを発揮します。処理は<InternalLink href="/rules/battle#cross-combo">戦闘ルールのクロスコンボ</InternalLink>で確認してください。
