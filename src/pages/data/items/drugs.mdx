# ドラッグ

<ImageBlock
  src="/images/data/items/drugs_hero.webp"
  alt=""
  loading="eager"
  width={heroImageDimensions.width}
  height={heroImageDimensions.height}
/>

「ヤク」の生き様を選んだキャラクターのみが使用できる特別なアイテムです。ドラッグを使用すると、<InternalLink href={`/rules/battle#${badStatusHeadingId}`}>バッドステータス</InternalLink>の「バッドトリップ」という、体力と精神力を失わせる状態異常が発生します。\
ドラッグは一度に複数使用することができますが、その分だけバッドトリップの影響が強くなってしまいます。バッドトリップは**BT**と省略して記載されることがあります。\
**精神力が0以下の場合は新たにドラッグを使用することはできません。**

<Callout type="warning">
ドラッグは短時間しか効果を発揮しないため、戦闘外では使用することができません。
</Callout>

## データの見方

<LegendContainer ariaLabel="ドラッグデータの見方">
  <DrugCard
    slot="card"
    name="①スピードドラッグ"
    credit="②2"
    timing="③SU"
    setQuantity="④3"
    badTripIntensity="⑤2"
    effect="⑥戦闘終了まで敏捷+2。"
  />

1. **名称**：そのドラッグの名称です。
2. **信用**：そのドラッグを取得するために必要な信用の数値です。
3. **使用タイミング**：戦闘中いつそのドラッグを使用できるタイミングです。詳しくは<InternalLink href={`/data#${skillTimingHeadingId}`}>スキルの見方の「タイミング」</InternalLink>を確認してください。同じタイミングのドラッグは複数同時に使用できますが、**同名のドラッグの効果は重複して発揮されません。**
4. **1セット数量**：そのドラッグ1つあたり、シナリオ中何回使用できるかを表します。同名のドラッグを複数取得した場合は、その数×1セット数量だけシナリオ中使用できます。
5. **BT強度**：そのドラッグを使用した際に受けるバッドトリップの強度です。
6. **効果**：ドラッグを使用した際、得られる効果です。
</LegendContainer>
