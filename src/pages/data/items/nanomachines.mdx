# ナノマシン

<ImageBlock src="/images/data/items/nanomachines_hero.webp" alt="" loading="eager" />

「スミ」の生き様を選んだキャラクターのみが使用できる特別なアイテムです。ナノマシンは体に入れ墨を入れることで使えるようになり、「頭」「胴体」「腕」「足」それぞれ1つ入れることができます。すべてのナノマシンは、指定された「発動精神力」点の精神力を消費することで、効果を発揮します。戦闘中は「セットアップ」タイミングで1つだけ発動することができます。効果欄に「武器化」と書かれている場合は、発動中同名の武器が使用できます。
ナノマシンの使用は肉体に負担がかかるため、**ナノマシンごとに設定された「埋め込み点数」の合計が「肉体」以下でなければなりません。**

<Callout type="warning">
ナノマシンは非戦闘中に発動させ、シーン中効果を得ることができます。ただし、ナノマシンを発動させると、周りのキャラクターに威圧感を与えてしまいます。戦闘中以外の場面において発動させると、その「点数」だけ非戦闘技能の判定数が減少します。
</Callout>

## ナノマシン埋め込みによる身体強化

ナノマシンを埋め込んでいる場合、最大体力が **「埋め込んでいるナノマシンの発動精神力のうち最大の値」** だけ増加します。スキル、アイテムの効果で発動精神力が減少したとしても、この値は減少しません。

    <li><strong>埋め込み点数</strong>：そのナノマシンが肉体に与える負荷です。埋め込んでいるナノマシンのこの値の合計が肉体以下でなければなりません。</li>
    <li><strong>効果</strong>：そのナノマシンを<strong>発動させているときに得られる効果です。</strong>同じナノマシンを複数発動させても効果は重複しません。</li>
  </ol>
</LegendContainer>
