# ナノマシン

「スミ」の生き様を選んだキャラクターのみが使用できる特別なアイテムです。ナノマシンは体に入れ墨を入れることで使えるようになり、「頭」「胴体」「腕」「足」それぞれ1つ埋め込むことができます。どの部位にどのナノマシンを埋め込むかは自由に決められます。\
すべてのナノマシンは、指定された「発動精神力」点の精神力を消費することで、効果を発揮します。戦闘中は「セットアップ」ごとに新たに1つだけ発動することができ、複数ラウンドかけて発動させる、あるいはスキルの効果によって複数のナノマシンを同時に発動させることもできます。発動したナノマシンの効果は、シーン終了時またはスキルの効果で解除されるまで持続し、任意に解除することはできません。\
効果欄に「武器化」と書かれている場合は、発動中同名の武器が使用できます。武器化ナノマシンは、通常の装備する武器とは別に、いくつでも戦闘中使用可能です。\
ナノマシンの使用は肉体に負担がかかるため、**ナノマシンごとに設定された「埋め込み点数」の合計が「肉体」以下でなければなりません。**

<Callout type="warning">
ナノマシンは非戦闘中に発動させ、シーン中効果を得ることができます。ただし、ナノマシンを発動させると、周りのキャラクターに威圧感を与えてしまいます。戦闘中以外の場面において発動させると、発動させているナノマシンの「埋め込み点数」の合計だけ非戦闘技能の判定数が減少します。
</Callout>

## ナノマシン埋め込みによる身体強化

ナノマシンを埋め込んでいる場合、最大体力が **「埋め込んでいるナノマシンの発動精神力のうち最大の値」** だけ増加します。スキル、アイテムの効果で発動精神力が減少したとしても、この値は減少しません。

## データの見方

<LegendContainer ariaLabel="ナノマシンデータの見方">
  <NanomachineCard
    slot="card"
    name="①マシラ"
    credit="②4"
    activationMentalCost="③15"
    implantPoints="④3"
    effect="⑤武器化。筋力+1。"
  />

1. **名称**：そのナノマシンの名称です。
2. **信用**：そのナノマシンを埋め込むのに必要な信用の数値です。
3. **発動精神力**：そのナノマシンを使用するために支払う精神力の点数です。
4. **埋め込み点数**：そのナノマシンが肉体に与える負荷です。埋め込んでいるナノマシンのこの値の合計が肉体以下でなければなりません。
5. **効果**：そのナノマシンを**発動させているときに得られる効果です**。同じナノマシンを複数発動させても効果は重複しません。
</LegendContainer>
