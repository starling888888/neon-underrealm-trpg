# はじめに

このページでは、本作を遊ぶ前に共有しておきたい基本用語とゴールデンルール、そしてルールサイトの読み進め方を案内します。

## 遊ぶために必要なもの

- ゲームを進行するGM（ゲームマスター）1人と、PCを担当するPL（プレイヤー）1人以上
- 10面体ダイスと6面体ダイスをそれぞれ10個以上。20個程度あると、より快適に遊べます
- このルールサイトと、キャラクターを記録するためのシートや筆記具
- 参加者どうしで物語を相談し、遊ぶ時間

本作は多数のダイスを使うため、オンラインセッションで遊ぶことをおすすめします。

## 基本的な用語

- **プレイヤー**：このゲームで後述するプレイヤーキャラクターを使って遊ぶ貴方がたのことです。PLと表現されることもあります。
- **ゲームマスター**：このゲームで物語をつくり、後述するノンプレイヤーキャラクターを操って物語を進める人物のことです。GMと表現されることもあります。
- **キャラクター**：ゲーム中に登場する人物たちのことです。
- **プレイヤーキャラクター**：プレイヤーの分身として貴方がたが生み出し、ゲーム中で活躍するキャラクターのことです。PCと表現されることもあります。
- **ノンプレイヤーキャラクター**：ゲーム中に登場する人物のうち、プレイヤーキャラクターではないキャラクターのことです。主にGMがそれらのキャラクターを操ります。NPCと表現されることもあります。
- **シナリオ**：一つの物語が始まり、終わるまでの単位です。このゲームで遊ぶこととはすなわちこのシナリオをGMとPLで紡ぎ出すことです。
- **キャンペーン**：複数のシナリオからなる長い物語です。シナリオとシナリオの間にはPCが成長していくことがあります。
- **D、サイコロ、ダイス**：サイコロのことを表します。このゲームでは10面体と6面体のサイコロを使用します。`nd10` と書かれた場合は10面体のサイコロをn個、`nd6` と書かれた場合は6面体のサイコロn個のことを指します。
- **判定**：なにか行動を起こした際にそれがどれだけうまくできたかをサイコロを振って決める行為です。
- **達成値**：判定においてどれだけうまくできたかを表す数値です。高ければ高いほどその結果が優れていることを示します。
- **判定数**：判定を行う際にふるサイコロの数です。判定を行う際に使用するサイコロは常に10面体サイコロです。判定に6面体サイコロは使いません。

<Callout type="warning" title="ゴールデンルール" titleHeadingLevel={2}>

ゲーム中に文中の表現や考えが矛盾していたり、ルールの適用が分からなくなったりすることがあります。そうしたときの指針がゴールデンルールです。複数のゴールデンルールが矛盾する場合は、より下にあるものを優先します。

1. **詳細は概要に勝る**: ルール本文とスキルやアイテムなどの詳細なデータが矛盾する場合は、より詳細に書かれた内容を優先します。
2. **受動は能動に勝る**: 能動側と受動側の達成値が同じ場合、受動側が勝ちます。効果が矛盾する場合も、受動側の効果を優先します。
3. **巻き戻し不可**: ルールの適用を間違えて本来と異なる結果が出ても、行った判定やデータの適用を改めて行いません。
4. **GMの判断に従う**: ルールやデータの解釈が分からない場合は、GMが扱いを決めます。PLはその裁定に従います。

割り算を行った際の数値は切り上げます。ただし、個別の説明で切り捨てると記載されている場合は、詳細な説明を優先します。

</Callout>
