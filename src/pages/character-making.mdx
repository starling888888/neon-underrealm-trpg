このページでは、セッションで使う仕事人を作るための基本要素と手順を説明します。キャラクターシートを用意し、GMや他のPLと相談しながら作成してください。

初めて仕事人を作るなら、決まった初期値で進められる**コンストラクション**がおすすめです。経験点を使い、流儀やスキルを自由に調整したい場合は**フルスクラッチ**を選びます。基本要素を先にすべて覚える必要はありません。下の作成手順を進めながら、必要な項目へ戻って確認してください。

## キャラクターの基本

仕事人は、次の要素を組み合わせて表します。

- **能力値**: どんな行動が得意かを表す数値。副能力値の算出にも使います。
- **流儀**: 仕事人としての戦い方。
- **生き様**: 科学技術を用いるか、用いないかを含む生き方。
- **スキル**: 戦闘中に使う技。流儀、生き様、共通の3種類を基本に、プライマリ流儀と生き様にはそれぞれボーナススキルがあります。
- **格**: 仕事人としてどれほどの存在かを表す数値。今後の成長でも参照します。
- **副能力値**: 体力、精神力、移動力など、戦闘で使う数値。
- **アイテム**: 武器、防具、生き様ごとの特別な装備。
- **技能**: 判定で使う能力。
- **縁**: 他のキャラクターとのつながり。

ここからは、それぞれをもう少し詳しく見ていきます。

### 能力値

能力値は、仕事人がどんな行動に長けているかを表します。副能力値の算出にも使います。

能力値は、プライマリ流儀の基礎能力値をもとに、生き様が示す能力値ポイントをプレイヤーが割り振って決めます。選ぶ流儀と生き様、作成時の能力値の割り振りによって、仕事人ごとに異なる値になります。

| 能力値 | 説明 |
| --- | --- |
| 筋力 | どれだけ力が強いか。 |
| 敏捷 | どれだけ素早く動けるか。 |
| 感覚 | 周囲の状況にどれだけ鋭敏か。 |
| 肉体 | どれだけ身体が強靭か。 |
| 精神 | どれだけ精神が頑強か。 |

### 流儀と生き様

**流儀**は、仕事人としての戦い方を表します。複数の流儀を修め、それぞれのレベルを上げることもできます。そのうち1つをプライマリ流儀として選びます。プライマリ流儀に応じて、キャラクターの特徴を定義する特別なプライマリボーナススキルを1つ得ます。このスキルは1レベルで効果を発揮し、成長させることはできません。

**生き様**は、どのような科学技術を用いて、あるいは用いずに生きているかを表します。選べる生き様は1つだけです。生き様を選ぶと、戦闘中のリソース獲得に役立ち、キャラクターの方向性を表す生き様ボーナススキルを1レベル取得します。生き様のレベルを成長させるときは、ほかの生き様スキルと同じように、生き様ボーナススキルも成長させられます。

流儀には、**体力増加値**と**精神力増加値**が設定されています。これは、プライマリ流儀のレベルに応じて最大体力と最大精神力へ加わる値です。生き様には、**初期体力係数**と**初期精神力係数**が設定されています。これらはそれぞれ肉体と精神に掛けて、最大体力と最大精神力を決める値です。

能力値、最大体力、最大精神力には、選んだ流儀と生き様に加え、能力値の割り振りやスキル取得といった**作成時の選択**が反映されます。

流儀と生き様の詳細は、<InternalLink href="/data/ryugi">流儀</InternalLink>と<InternalLink href="/data/ikizama">生き様</InternalLink>を確認してください。

### 格

格は、仕事人としてどれほどの存在かを表す数値です。プライマリ流儀のレベルと生き様のレベルを合計して決めます。格は、今後キャラクターを成長させるときに参照します。

共通スキルの合計レベルは、格の半分までです。割り算の端数は、ゴールデンルールに従って切り上げます。

### スキル

スキルは、仕事人が戦闘中にどのような行動を取れるかを表します。スキルにはレベルと上限があり、上限を超えて成長させることはできません。

| 種別 | 取得方法 |
| --- | --- |
| 流儀スキル | 流儀が1レベル上がるごとに、1つ取得するか、取得済みスキルを1レベル上げます。 |
| プライマリボーナススキル | プライマリ流儀を選ぶと1レベル取得します。キャラクターの特徴を定義する特別なスキルであり、1レベルで効果を発揮し、成長させることはできません。 |
| 生き様スキル | 生き様が1レベル上がるごとに、1つ取得するか、取得済みスキルを1レベル上げます。 |
| 生き様ボーナススキル | 生き様を選ぶと1レベル取得します。戦闘中のリソース獲得に役立ち、キャラクターの方向性を表すスキルです。生き様のレベルを成長させるときは、ほかの生き様スキルと同じように成長させられます。 |
| 共通スキル | 経験点を支払って取得・成長させます。 |

共通スキルの詳細は、<InternalLink href="/data/common-skills">共通スキル</InternalLink>を確認してください。

### 副能力値

副能力値は、仕事人がどれだけ攻撃に耐えられるか、どれだけ素早く行動できるかなどを表します。能力値、流儀、スキル、アイテムの効果によって変わります。最大体力と最大精神力は後述の作成手順で決めます。

| 副能力値 | 説明 |
| --- | --- |
| 体力 | どれだけ攻撃に耐えられるかを表します。最大値は、体力増加値、初期体力係数、肉体、スキルの修正で決まります。 |
| 精神力 | 一部のスキルの使用に用います。最大値は、精神力増加値、初期精神力係数、精神、スキルの修正で決まります。 |
| 移動力 | どれだけの距離を移動できるかを表します。`4 + 敏捷 ÷ 5（端数切り上げ）+ アイテム・スキルによる修正`で決めます。 |
| 行動値 | どれだけ素早く行動できるかを表します。`敏捷 + 感覚 × 2 + アイテム・スキルによる修正`で決めます。 |
| 防御力 | 物理的なダメージをどれだけ減らせるかを表します。アイテム・スキルによって決まります。 |
| ダメージ軽減 | 特殊な攻撃を含むダメージをどれだけ減らせるかを表します。`アイテム・スキルによる修正`で決めます。 |

### アイテム

仕事人は**信用**を持ち、アイテムを使うには、個々のアイテムに設定された必要信用の合計が所持信用以下でなければなりません。

武器と防具には、ただ持っていることを表す「所持」と、戦闘で使える状態を表す「装備」があります。アイテムの詳細は<InternalLink href="/data/items">アイテム</InternalLink>を確認してください。

| 種別 | 説明 |
| --- | --- |
| 武器 | 戦闘中の攻撃に使うアイテムです。特別な場合を除き、装備できるのは1つです。 |
| 防具 | 主に防御力とダメージ軽減に影響するアイテムです。所持できるのは1つで、その防具を装備しているものとします。 |

### 技能

技能は、判定を行うときに使います。GMから指定された技能や、戦闘中のタイミングで決められた技能を用いて判定します。戦闘リアクションは、攻撃種別ごとに決められた能力値の中から1つを選びます。

#### 戦闘技能

戦闘技能は主に戦闘中に使います。喧嘩、暗殺、発砲、格闘、干渉を能動技能、防御、回避、耐え、抵抗を受動技能と呼ぶことがあります。

| 技能 | 対応能力値 | 説明 |
| --- | --- | --- |
| 喧嘩 | 筋力 | 大きな武器で攻撃する技能です。 |
| 暗殺 | 敏捷 | 小さな武器で攻撃する技能です。 |
| 発砲 | 感覚 | 離れた敵を攻撃できる武器で攻撃する技能です。 |
| 格闘 | 肉体 | 素手や素足など、身体を使って攻撃する技能です。 |
| 干渉 | 精神 | 特殊能力で攻撃する技能です。 |
| 防御 | 筋力・敏捷・感覚 | 近接攻撃、射撃攻撃のダメージを減らす技能です。判定時にいずれかの能力値を選びます。 |
| 回避 | 筋力・敏捷・感覚 | 近接攻撃、射撃攻撃を無効化する技能です。判定時にいずれかの能力値を選びます。 |
| 耐え | 肉体・精神 | 特殊攻撃に耐える技能です。判定時にいずれかの能力値を選びます。 |
| 抵抗 | 肉体・精神 | 特殊攻撃に抵抗する技能です。判定時にいずれかの能力値を選びます。 |

#### 非戦闘技能

非戦闘技能は主に戦闘以外で使います。戦闘中に重要な効果を発揮するものもあります。

| 技能 | 対応能力値 | 説明 |
| --- | --- | --- |
| 脅迫 | 筋力 | 人を脅して情報を得たり、口封じをしたりする技能です。 |
| 力比べ | 筋力 | 他人と競ったり、壁を壊したりする技能です。 |
| 根性 | 筋力 | 痛みやストレスに耐える技能です。 |
| 偵察 | 敏捷 | 密かに話を聞いたり、覗き見たりする技能です。 |
| 軽業 | 敏捷 | 高い場所から安全に飛び降りる技能です。 |
| 手業 | 敏捷 | 人のものを盗み取る技能です。 |
| イカサマ | 感覚 | 人を欺く技能です。手品なども含みます。 |
| 危険察知 | 感覚 | 迫る危険に事前に気付く技能です。 |
| 分析 | 感覚 | 情報を読み解き、分析する技能です。 |
| 運転 | 肉体 | 車やバイクを運転する技能です。 |
| 生存 | 肉体 | 野外活動などを行う技能です。 |
| 仁義 | 肉体 | 自分の信念や約束を貫く技能です。 |
| 賭博 | 精神 | 賭け事を行う技能です。 |
| 交渉 | 精神 | 人と話して情報を得る技能です。 |
| ハッキング | 精神 | コンピュータなどを操作する技能です。 |

### 縁

縁は、キャラクターどうしのつながりを表します。縁は次の3つの要素で記録します。

- **対象**: 誰と縁を結んでいるか。
- **種別**: その対象にポジティブな感情を持つか、ネガティブな感情を持つか。
- **関係**: 対象との具体的な関係や、抱いている感情。

## 初期縁の決定（RoC方式）

キャラクター作成時には、必ず1つ初期縁を結びます。初期縁一覧から選ぶか、サイコロで決めてください。コンストラクションでもフルスクラッチでも、この手順を行います。

サイコロで決める場合は、10面体ダイスを2回振ります。1回目の出目で対象を決め、2回目の出目が奇数ならポジティブ、偶数ならネガティブな感情を選びます。

初期縁一覧の文章を手掛かりに、自分なりの関係を決めてキャラクターシートへ記録してください。可能なら、対象の名前や設定も決めておくと、セッション中の会話や判断に使いやすくなります。

## コンストラクション

コンストラクションは、プライマリ流儀を1つ、生き様を1つ選んでキャラクターを作る方法です。

### 1. 流儀と生き様を決める

プライマリ流儀と生き様を決めます。初期作成では、プライマリ流儀は4レベル、生き様は2レベルです。

### 2. 能力値を決める

選んだプライマリ流儀の基礎能力値に、生き様で決まる4つの能力値ポイントを割り振ります。たとえば能力値ポイントが `5、4、4、1` なら、それぞれが能力値へ加える数値です。4つの異なる能力値を選び、各能力値へ4つの値のどれか1つを割り振ってください。同じ能力値へ複数の値を割り振ることはできません。

<Callout type="tip" title="能力値の配分">

配分に迷ったら、まずプライマリ流儀で使いたい能力値を10程度まで伸ばしてください。その後、肉体と精神を高めると、体力や精神力を確保しやすくなります。

</Callout>

<Callout type="example" title="ケンカヤとケジメを選んだ場合">

<KenkayaAttributesExample />

</Callout>

### 3. スキルを取得する

プライマリ流儀スキルを4レベル、生き様スキルを2レベル分取得します。流儀スキル4レベルとは別に、キャラクターの特徴を定義する特別なプライマリボーナススキルを1レベル取得します。プライマリボーナススキルは1レベルで効果を発揮し、成長させることはできません。

生き様ボーナススキルも1レベル取得します。これは戦闘中のリソース獲得に役立ち、キャラクターの方向性を表すスキルです。生き様を成長させるときは、ほかの生き様スキルと同じように、生き様ボーナススキルも成長させられます。

共通スキルは、合計2レベル分取得します。

### 4. 共通スキルボーナスを取得する

コンストラクションでは、共通スキルを合計2レベル取得しているため、プライマリ流儀に応じた追加ボーナスを得ます。

<RyugiCommonSkillBonusTable level={2} ryugiList={ryugiList} />

「判定数+1」などの数値がルール上で何を意味するかは、<InternalLink href="/rules">ルール</InternalLink>セクションで説明します。

### 5. 最大体力と最大精神力を決める

最大体力と最大精神力は、次の式で決めます。

| 項目 | 計算式 |
| --- | --- |
| 最大体力 | プライマリ流儀の体力増加値 × 4 + 生き様の初期体力係数 × 肉体 + スキルの修正 |
| 最大精神力 | プライマリ流儀の精神力増加値 × 4 + 生き様の初期精神力係数 × 精神 + スキルの修正 |

<Callout type="example" title="ケンカヤとケジメを選んだ場合">

<KenkayaHealthExample level={4} />

取得したスキルに修正がある場合は、この値へ加えます。

</Callout>

### 6. アイテムを取得する

アイテムに設定された信用の合計が10点以下になるように、アイテムを取得します。

### 7. 得意技能を決める

非戦闘技能のうち、どれが得意かを決めます。次のどちらかを選んでください。

- 同じ対応能力値の技能を2つと、それ以外の対応能力値の技能を1つ選ぶ。
- それぞれ異なる対応能力値の技能を4つ選ぶ。

得意技能に選んだ非戦闘技能を用いる判定では、対応能力値を2倍にして判定数を決めます。そのため判定ダイスが大幅に増え、成功しやすくなります。選んだ技能は、キャラクターシートの得意技能欄へ記録してください。

## フルスクラッチ

フルスクラッチは、経験点を使って自由にキャラクターを作る方法です。

### 1. プライマリ流儀と生き様を決める

プライマリ流儀と生き様を選びます。プライマリ流儀と生き様の初期1レベルは無料です。プライマリ流儀に選んだ流儀は、後述する成長によるスキル取得とは別に、キャラクターの特徴を定義する特別なプライマリボーナススキルを1つ取得します。このスキルは1レベルで効果を発揮し、成長させることはできません。

生き様を選ぶと、生き様ボーナススキルを1レベル取得します。これは戦闘中のリソース獲得に役立ち、キャラクターの方向性を表すスキルです。

<h3 data-anchor-id="full-scratch-ability-values">2. 能力値を決める</h3>

能力値はコンストラクションと同じ方法で決めます。

### 3. 経験点で成長させる

50点の経験点を使い、次の表に従って自由に成長させます。プライマリ流儀と生き様は初期1レベルを無料で得ます。プライマリ流儀以外の流儀も、経験点を支払って取得・成長させられます。流儀と生き様は、1レベル取得・成長させるごとに、1レベル分のスキルを取得できます。生き様を成長させるときは、ほかの生き様スキルと同じように、生き様ボーナススキルも成長させられます。

共通スキルも自由に取得・成長させられます。ただし、共通スキルの合計レベルは格の半分までです。格はプライマリ流儀のレベルと生き様のレベルの合計で決まります。割り算の端数は、ゴールデンルールに従って切り上げます。

<Callout type="warning" title="注意">

コンストラクションに記載された共通スキルボーナスを得られるのは、共通スキルを合計2レベル以上取得している場合だけです。共通スキルが0レベルまたは1レベルの場合は、このボーナスを得られません。

</Callout>

| 対象 | 必要経験点 |
| --- | --- |
| 流儀 | 1レベルごとに10点 |
| 生き様 | 1レベルごとに10点 |
| 共通スキル | 1レベルごとに5点 |

<Callout type="example" title="経験点配分の例">

ケンカヤをプライマリ流儀として3レベル、カベを1レベル、ケジメを2レベル、共通スキルを2レベル取得する場合を考えます。ケンカヤとケジメの初期1レベルは無料です。

| 対象 | 計算 | 使用経験点 |
| --- | --- | --- |
| ケンカヤ（プライマリ） | (3 - 1) × 10 | 20点 |
| カベ | 1 × 10 | 10点 |
| ケジメ | (2 - 1) × 10 | 10点 |
| 共通スキル | 2 × 5 | 10点 |
| 合計 |  | 50点 |

</Callout>

### 4. 最大体力と最大精神力を決める

最大体力と最大精神力は、次の式で決めます。

| 項目 | 計算式 |
| --- | --- |
| 最大体力 | プライマリ流儀の体力増加値 × レベル + 生き様の初期体力係数 × 肉体 + スキルの修正 |

### 6. 得意技能を決める

得意技能の決定は、コンストラクションと同じ方法で行います。

得意技能に選んだ非戦闘技能を用いる判定では、対応能力値を2倍にして判定数を決めます。そのため判定ダイスが大幅に増え、成功しやすくなります。選んだ技能は、キャラクターシートの得意技能欄へ記録してください。
