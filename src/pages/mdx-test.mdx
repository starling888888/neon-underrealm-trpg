# MDX表示確認

Markdown本文として見出しや段落を書けることを確認するための最小ページです。

## 見出しとリスト

- 箇条書きの余白を確認します。
- 長めの文章でも行間が詰まりすぎないことを確認します。
- 落ち着いたmarkerが本文を邪魔しないことを確認します。

> 引用や補足文の表示確認です。本文と区別しつつ、強すぎない見た目にします。

## コードと表

インラインコードは `skill_check` のように表示します。

```ts
const target = 16;
const total = roll + ability + skill;
```

| 項目 | 説明 | 備考 |
|---|---|---|
| 判定 | 行動の成否を決める処理です。 | 横幅確認用 |
| 補足 | 長文の読みやすさを確認します。 | 密度確認用 |
